# Portfolio Website

A minimal React portfolio website with a lofi aesthetic.

## Getting Started

1. Install dependencies:
```
npm install
```

2. Start the development server:
```
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Structure

- **Landing Page**: Simple grey background with "View My Work" button
- **My Projects**: Grid layout with placeholder project cards
- **About Me**: Text content with scrolling icon animations above and below
